# Game Scoreboard using MAX7219

This projects utilizes a MAX7219 to control a 8 by 8 LED matrix!

## Materials
- 20 - 30 jumper wires
- MAX7219CNG
- 8 by 8 LED matrix
- Arduino Uno
- Breadboard
